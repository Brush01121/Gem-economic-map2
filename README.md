# 🌍 GEM (Geo Econo Map)

경제 데이터 시각화 웹앱

## 📌 프로젝트 소개
세계 각국의 경제 지표(GDP, 빈곤지수, 실업률, 인플레이션)를 시각화하여 보여주는 웹 애플리케이션입니다.

## 🛠️ 기술 스택
- **Google Apps Script** - 백엔드
- **Claude AI API** - 경제 데이터 분석
- **HTML/CSS/JavaScript** - 프론트엔드
- **SVG** - 지도 시각화

## ✨ 주요 기능
- 📊 4가지 경제 지표 선택 (GDP, 빈곤지수, 실업률, 인플레이션)
- 🌍 10개 주요 국가 데이터 제공
- 🤖 Claude AI 기반 실시간 데이터 분석
- 🎨 깔끔하고 현대적인 UI/UX

## 🚀 설치 방법

### 1. Google Apps Script 설정
1. [Google Apps Script](https://script.google.com) 접속
2. 새 프로젝트 생성
3. `code.gs` 파일에 코드 복사-붙여넣기
4. HTML 파일 추가 (`Index.html`)
5. 웹앱으로 배포

### 2. Claude API 키 발급
1. [Anthropic Console](https://console.anthropic.com/) 접속
2. API 키 발급
3. 앱 설정에서 API 키 입력
